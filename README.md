# 🔐 Password Vault Python

A beginner-friendly Password Vault built using Python.

This project allows users to manage account credentials through a simple menu-driven interface.

## Features

- User Signup
- User Login
- Add New Account Credentials
- View Saved Accounts
- Search for Existing Accounts
- Store Data Using Text Files

## Concepts Used

- Python Basics
- Conditional Statements
- Loops
- User Input
- File Handling
- String Operations

## Files Used

- `users.txt` → Stores user signup information
- `passwords.txt` → Stores account credentials

## Future Improvements

- Password Validation
- Secure Login System
- Password Generator
- Update/Delete Credentials
- GUI Version using Tkinter
